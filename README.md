# Система управління духовним центром «Нове Життя»

Професійна ERP-система для реабілітаційних центрів, розроблена для координації служіння, моніторингу прогресу підопічних та управління внутрішніми ресурсами.

## Основні модулі

- **Дашборд**: Огляд активних завдань, звітів та сповіщень.
- **Підопічні**: Повна база даних з історією прогресу, контактами та духовними звітами.
- **Чат**: Канали для спілкування між наставниками та служителями.
- **Бібліотека**: Репозиторій навчальних матеріалів, відео та документації (Духовність, Психологія, Залежність).
- **Графік**: Автоматизоване управління чергуваннями та завданнями.
- **Фінанси**: Прозорий моніторинг доходів та витрат (доступ обмежено).

## Технологічний стек

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Firebase (Firestore, Auth).
- **PWA**: Підтримка встановлення на Android як нативний додаток.
- **Локалізація**: Повна підтримка української мови.

## Встановлення (PWA)

Додаток оптимізовано для мобільних пристроїв Android. Для встановлення:
1. Відкрийте додаток у Chrome.
2. Натисніть "Додати на головний екран".
3. Ви отримаєте іконку додатку в меню.

## Безпека

Система використовує рольову модель доступу (RBAC):
- **Admin**: Повний доступ.
- **Senior Mentor**: Розширений доступ до фінансів та графіків.
- **Mentor**: Перегляд підопічних та участь у чаті.
